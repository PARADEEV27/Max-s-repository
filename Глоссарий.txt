13.09.2025 - Начал вести Глоссарий, зашёл в аккаунт github

GitHub — это
онлайн-платформа и сервис для хостинга проектов, основанный на системе управления версиями Git. Он позволяет разработчикам хранить, управлять и совместно работать над кодом и другими файлами, отслеживать изменения, работать в команде над общим проектом и делиться своими наработками. GitHub также служит своего рода социальной сетью для программистов, где можно находить проекты с открытым исходным кодом, взаимодействовать с ними и демонстрировать свое портфолио.
